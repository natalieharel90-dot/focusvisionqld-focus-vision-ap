**What this is for:** Getting back into the staff dashboard when sign-in fails, working through the most common causes first.

Staff sign in with three things: your email, your password, and a 6-digit code from your authenticator app. The authenticator app is a free app on your phone that shows a code that changes every 30 seconds. A problem with any one of the three will stop you.

## Things to try, in order

1. Check you typed your email and password correctly, with no extra spaces.
2. Make sure Caps Lock is off — passwords are case-sensitive.
3. If the password is rejected, select **Forgot password** and follow the reset email.
4. For the 6-digit code, open your authenticator app and type the code that is showing **right now**.
5. If the code is rejected, wait for the app to refresh and enter the new code straight away.
6. Check that your phone's clock is set to update automatically — a wrong clock makes every code fail.
7. Try a different browser or a private window to rule out a stuck saved login.

:::watch-out
After several failed attempts your account may be locked for a short period. Wait 15 minutes, then try again.
:::

:::tip
The code is time-based, so it only works for about 30 seconds. Type it quickly rather than copying it slowly.
:::

If you have lost or replaced your phone, your authenticator codes are gone. Ask your clinic manager to reset two-factor for your account so you can set it up again.
