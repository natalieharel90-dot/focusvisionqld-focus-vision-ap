**What this is for:** Setting up the default medications and appointments that a patient receives the moment they are added to the system.

## What a procedure template is

A **procedure template** is a saved preset for one combination of surgeon and procedure — for example, "Dr Chen · LASIK". Each combination has exactly one template. The template holds the default eye drops, tablets, and follow-up appointments that most patients of that surgeon and procedure need.

When you set up a new patient, applying the template **materialises** those items — it creates real medications and appointments on the patient's record automatically, so you do not type them in by hand.

## How to create one

1. Open the **Procedures library** from the sidebar.
2. Click **Add template**.
3. Choose the surgeon, then choose or type the procedure name.
4. Click **Continue** to open the template editor.
5. Add each default medication with its dose, route, frequency, and dosing times.
6. Add each follow-up appointment with how many days after surgery it falls.
7. Optionally link a routing ruleset, then click **Save template**.

:::tip
You can type a procedure name that is not in the list. The clinic can offer any procedure — it is added automatically when you save.
:::

:::watch-out
Editing a template only affects **future** patient set-ups. Patients already onboarded keep the meds and appointments they were given.
:::

:::patient-sees
Nothing directly — the template is staff-internal. Patients only see the medications and appointments once a template is applied to their record.
:::
