**What this is for:** Deciding whether a situation calls for a one-to-one reply or a single broadcast to many patients at once.

## The difference

A **bulk push** sends one message or content item to a filtered group of patients — a cohort — in a single action. An **individual message** is a one-to-one reply in a patient's thread in the shared inbox.

Both land in the patient's message thread, labelled "From Focus Vision team". The difference is reach and intent.

## When to use bulk push

Use bulk push when the same information helps a whole group:

- A reassurance note for everyone three to seven days after LASIK who often notice halos.
- A reminder about an upcoming public holiday closure for all active patients.
- A new educational video relevant to one procedure type.

## When to use an individual message

Use a one-to-one message when the content is specific to one person:

- Replying to a question a patient has asked.
- Following up after a flagged check-in.
- Anything that mentions that patient's own results or symptoms.

:::tip
If you find yourself sending the same wording to several patients in a row, that is a sign a bulk push would be faster and more consistent.
:::

:::watch-out
Bulk push is for general, reassuring information. Never use it to deliver anything that reads as personalised clinical advice — that belongs in a one-to-one message.
:::

:::patient-sees
Either way, the patient receives a message in their app thread. A bulk push looks the same to them as a normal message from the clinic.
:::
